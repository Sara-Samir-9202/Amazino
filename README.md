# 🛒 Amazino – E-Commerce Multi-Actor Platform

A modern e-commerce platform built with role-based access for Customers, Sellers, and Admins.  
Designed with a responsive UI, dynamic product management, shopping cart functionality, and interactive dashboards.

🌐 **Live Demo:** https://amazino-orcin.vercel.app/

---

# ✨ Features

## 👤 Customer Features
- Browse products with responsive UI
- Product details page
- Search and filtering system
- Shopping cart management
- Checkout flow
- Order history
- Authentication system

---

## 🛍️ Seller Features
- Add/Edit/Delete products
- Product management dashboard
- Order processing
- Sales analytics

---

## 🛡️ Admin Features
- User management
- Product moderation
- Dashboard analytics
- Platform monitoring
